# Future image filenames

Binary image files are intentionally **not committed** in this repository.
When you are ready to add real media, place JPG files at these paths and the HTML/CSS already documents where each one belongs:

- `assets/img/st-logo.jpg` — school logo for the header.
- `assets/img/image1.jpg` — vision/about hero image.
- `assets/img/image2.jpg` — horizontal strip image.
- `assets/img/image3.jpg` — aims/objectives image.
- `assets/img/image4.jpg` — activities/gallery image.
- `assets/img/image5.jpg` — library/classroom image.
- `assets/img/image6.jpg` — laboratory/computer-lab image.

Until those files are added, the site renders CSS-only placeholders so there are no binary assets in the repo.
